EN:

How to install TICQClient:
0) Delete previous component TICQClient if you installed it
1) Open TICQClient.dpk in Delphi
2) Goto Menu->Tools->Environment Options->Library
3) Press button "..." near Library path
4) Add path to TICQClient directory
5) Return to the window of dpk
6) Press Compile
7) Press Install
8) Now you can use component ICQClient on tab Samples

You can get newer version here: http://progs.volkov.spb.su/ticqclient/

RU:

Как установить TICQClient:
0) Удалите предыдущий компонент TICQClient, если ставили его
1) Откройте TICQClient.dpk в Delphi
2) Перейдите в Menu->Tools->Environment Options->Library
3) Нажмите кнопку "..." напротив Library path
4) Добавьте путь в папку, где лежит TICQClient
5) Вернитесь к окошку открытого dpk
6) Нажмите Compile
7) Нажмите Install
8) Теперь можете использовать компонент ICQClient на вкладке Samples

Новую версию можете взять здесь: http://progs.volkov.spb.su/ticqclient/
